## Abstract

Animals don't move by stringing together independent actions; they fluidly switch between behavioral motifs, each governed by its own dynamics. Premotor and basal ganglia circuits select which behavior to initiate and set its initial conditions, while the primary motor cortex evolves that behavior forward as a dynamical system. Yet computational approaches to modeling naturalistic behavior remain fragmented. While probabilistic state-space models (SSMs) can segment discrete behavioral syllables from keypoint data and reinforcement learning (RL) drives lower-level physics control, there exist no principled bridge between them. We introduce **Code2Act**, a **Switching Nonlinear Dynamical System (S-nLDS)** that unifies these levels within a single imitation framework. Built on observables from [MIMIC-MJX](https://arxiv.org/abs/2511.20532) (Zhang & Yang et al. 2025), a biomechanical simulation of naturalistic rodent behavior, our model replaces the linear regime dynamics of classical switching linear dynamical systems with a RNN trained in RL environment, capable of capturing the nonlinear, non-Gaussian structure of real motor control. The result is a system that jointly segments continuous behavior into discrete semantic units and generates long, biomechanically realistic motor sequences, closing the loop between behavioral understanding and motor generation in a way that mirrors the hierarchical dynamical organization of biological motor control.

</HighlightedSection>

## Problem Statement

**Probabilistic State Space Models** (PSSMs) model the dynamics of the trajectories well. They model the switching and segmental behaviors, but they fail to serve as a generative model of trajectories.

**Reinforcement Learning** (RL) are widely used for low-level motor control, but the learned latent of an end-to-end imitation model is not used for regeneration.

Current models capture what behavior looks like, but none can generate it with discrete syllable-level control.

## Pipeline Overview

Code2Act uses a **prior distillation** pipeline where a pre-trained encoder-decoder VAE provides the action generation backbone, and a GRU-based RNN learns to predict the encoder's latent distribution from discrete syllable codes.

<Figure>
  <Picture slot="figure" src={pipelineFig} alt="Code2Act prior distillation pipeline" />
  <Fragment slot="caption">
    **Code2Act Pipeline.** Keypoint-MoSeq segments reference trajectories into discrete syllable codes. A GRU-based RNN decoder takes (code embedding, proprioception) as input and produces hidden states. A unified VAE head maps the hidden state to a latent distribution (mu, sigma), from which z is sampled and fed through the pre-trained VAE decoder to produce motor commands. During training, a frozen pre-trained encoder provides KL alignment targets.
  </Fragment>
</Figure>

## Results

### Training Performance and Reward Decomposition

Code2Act achieves competitive imitation reward with the oracle VAE (Mimic-MJX) that has access to the full reference trajectory. The reward decomposition shows strong performance on coarse (root position/orientation) and fine (joint/end-effector) tracking.

### Behavior Instantiation

Code sequences are **instructional**: providing walk codes produces walking behavior, groom codes produce grooming, and rear codes produce rearing. The ghost-body videos below show multiple trajectories overlaid, all driven by the same behavioral code sequence but from different initial conditions.

<Video src="behavior_triptych_low.mp4" />

**Walk Pose Instantiation.** K=6 trajectories per behavior (Groom | Walk | Rear), all initialized from a walk-like starting pose. Each colored body is an independent rollout driven by the same KPMS code sequence. The decoder produces behavior-appropriate dynamics for each syllable.

### Code Sequence Displacement

Code sequences produce **interpretable kinematics**: walk codes drive horizontal (X-Y) displacement while groom and rear codes produce minimal locomotion. Vertical (Z) displacement distinguishes rearing from other behaviors.

### Generative Code Sequences

Code2Act turns behavior generation into a **code sequencing** problem. We fit generative models (empirical transition matrix, HMM, ARHMM) to the KPMS code sequences and use them to generate novel code sequences. These generated sequences produce coherent free-loop behavior without any reference trajectory.

<Video src="code_generation_triptych.mp4" />

**Free-loop generation.** Novel code sequences sampled from learned generative models (Transition Matrix | HMM | ARHMM) produce coherent whole-body behavior. No reference trajectory is provided — the decoder autonomously translates the generated code sequence into motor commands.

## Conclusion

**Code2Act** preserves the internal dynamics of each discrete syllable, turning naturalistic behavior generation into a problem of **code sequencing** rather than repeated interaction with a physics engine.

- **(1)** An engineering solution for **planning** naturalistic behavioral sequences without relying on physics.
- **(2)** A generative modeling framework, akin to fitting an ARHMM, for synthesizing whole-body behavior.

Code2Act bridges the gap between probabilistic state space models (which capture behavioral structure) and reinforcement learning (which produces motor control), by using discrete syllable codes as the interface between structure and action.
